The Fatburg

I read about something called a "fatburg". It's essentially a large pile of non-biodegradeable "stuff" that collects in the sewage system and clogs the pipes. Eventually, somebody has to go in and clean it all up. It makes me think of the bandaid-workarounds I've been using to create this project. The further I progress with the project, the more I see the need to go ahead and clean the pipes!
